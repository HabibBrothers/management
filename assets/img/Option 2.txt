Admin:

1. Creates user login credentials.

2. Uploads/Deletes PDF forms containing name, date and signature field.

3. Can download/delete received files in the folder called Inbox. 

User:

1. Sees documents name and email field.

2. User pastes (typing NOT allowed) email address and clicks send button. User sees Sent message.

3. User can see/search/download received files in an outside email account where the system automatically forwards all incomg files.

 
Client:

1. Receives an email with a link (email text below).

2. Client types name on the document, signs it in signature field (in browser Chrome, IE, Safari Etc.) and clicks Send. Dates on the document populates automatically. 

3. Client sees Thank you message after submission.
